Site Goals

This site was made to help expand the Disc Golf community. There are plenty of courses out in locations that not everyone knows about. Making this site allows the community to have a list of the courses and to give reviews about what the users think of the course. Each course will have have how many holes are in the course, what the type of tees the course uses, and the city location. By doing this, we hope to make it so new courses are discovered and everyone who is looking for new courses to try out will enjoy new locations. Disc Golf is a fun sport and is growing more popular every year, so this site is aimed to expand with the ever growing Disc golf population.



Features in this site includes adding new courses to the website. Each user can add courses to the website to expand the ever growing list of courses. Each user can also submit reviews on the courses, letting other users know what they think of the courses and any thoughts on how the courses can improve to make them more user friendly.
